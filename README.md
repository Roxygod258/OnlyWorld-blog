# OnlyWorld 个人博客

一个无需安装依赖、可以直接在本地打开的个人博客。内容包括学习笔记、杂谈、个人主页和可自定义欢迎页。

## 打开网站

直接双击 `index.html`。修改文件后，在浏览器中按 `Ctrl + F5` 强制刷新。

## 两种修改方式的区别

### 页面内修改

适合目前只在自己的电脑上浏览：

- 欢迎页：点击铅笔按钮。
- 个人资料：点击左上角头像进入个人主页，再点击右上角铅笔按钮。
- 整页壁纸和透明度：点击顶部半圆按钮。

这些设置保存在当前浏览器中，不会自动写回项目文件。清除浏览器网站数据后，本地修改会消失；把网站发布到网上后，其他访客也看不到这些本地设置。

### 修改内容文件

适合以后公开发布。文件中的内容会成为所有访客看到的默认内容：

- `site-content.js`：欢迎页和个人资料。
- `site-posts.js`：学习笔记和杂谈文章。
- `styles.css`：颜色、字号和布局。

建议使用记事本或代码编辑器打开文件，并以 UTF-8 编码保存。

## 修改个人资料

### 在页面内修改

1. 点击左上角的 `O` 头像。
2. 在个人主页右上角点击铅笔按钮。
3. 可以修改头像、主页标题、个人简介、邮箱、GitHub 用户名、当前方向、记录主题和学习状态。
4. 点击“保存”。

头像会被自动压缩并保存在当前浏览器中。

### 修改公开默认资料

打开 `site-content.js`，修改 `profile` 内引号中的内容：

- `avatar`：头像路径，例如 `"assets/avatar.jpg"`。
- `greeting`：个人主页标题。
- `bio`：个人简介。
- `email`：邮箱地址。
- `github`：GitHub 用户名，不需要填写完整网址。
- `direction`：当前学习方向。
- `topics`：主要记录主题。
- `status`：当前状态。
- `website`、`belief`：个人主页下方的介绍文字。

公开头像的设置方法：

1. 把头像图片放入 `assets` 文件夹，例如 `assets/avatar.jpg`。
2. 将 `site-content.js` 中的 `avatar: ""` 改成 `avatar: "assets/avatar.jpg"`。
3. 保存文件并刷新网页。

如果之前在页面里保存过个人资料，本地设置会优先于文件内容。进入个人资料编辑面板，点击“恢复默认”，即可重新显示 `site-content.js` 中的内容。

## 修改欢迎页

点击欢迎页或顶部的铅笔按钮，可以修改欢迎页文字。要修改公开默认文字，请编辑 `site-content.js` 中的 `welcome`：

- `kicker`：标题上方的小字。
- `title`：主标题。
- `subtitle`：欢迎语。
- `button`：进入按钮文字。

如果文件修改后没有变化，在欢迎文字编辑面板中点击“恢复默认”，清除当前浏览器保存的覆盖内容。

## 添加学习笔记

打开 `site-posts.js`，找到 `notes: [`。复制其中一整段文章对象，粘贴到同一数组中，再修改内容：

```js
{
  id: "unique-note-id",
  title: "文章标题",
  date: "2026-07-17",
  tags: ["标签一", "标签二"],
  summary: "显示在文章列表中的摘要。",
  readTime: "5 分钟",
  content: `
    <p>第一段正文。</p>
    <h2>小标题</h2>
    <p>第二段正文。</p>
  `,
},
```

注意：

- `id` 必须是唯一的英文标识，不能和其他文章重复。
- 日期必须使用 `YYYY-MM-DD` 格式。
- 正文放在一对反引号之间。
- 每篇文章结束的 `},` 不要删除。
- 段落使用 `<p>文字</p>`，小标题使用 `<h2>标题</h2>`。
- 代码块可以使用 `<pre><code>代码</code></pre>`。

网站会自动更新首页目录、搜索和日期归档，不需要修改其他文件。

## 添加杂谈

打开 `site-posts.js`，找到 `thoughts: [`，复制一篇现有杂谈并修改。杂谈必须保留：

```js
kind: "thought",
```

其他字段与学习笔记相同。保存后，文章会自动显示在杂谈区、侧栏目录、搜索和日期归档中。

## 自定义壁纸

点击顶部半圆按钮，可以上传一张整页壁纸，并调节浅色遮罩、背景柔化、内容不透明度和导航不透明度。每位访客的设置都保存在各自浏览器中。

若要设置公开默认壁纸，可以把图片放入 `assets` 文件夹，再在 `styles.css` 的 `.wallpaper-main` 中修改 `background-image`。

公开默认透明度位于 `styles.css` 顶部：

- `--content-opacity: 0.70`：正文和目录毛玻璃不透明度。
- `--nav-opacity: 0.88`：顶部导航不透明度。

## 添加照片

把照片放入 `assets` 文件夹，然后在 `site-content.js` 的 `photos` 数组中添加：

```js
photos: [
  {
    src: "assets/photo-01.jpg",
    alt: "照片内容描述",
    caption: "照片标题",
    date: "2026-07-17",
  },
],
```

## 设置网易云歌单

本地设置：打开网站的“音乐”页面，在输入框中填写网易云歌单 ID，然后点击“载入歌单”。

公开默认设置：打开 `site-content.js`，修改：

```js
music: {
  title: "我的网易云歌单",
  neteasePlaylistId: "123456789",
},
```

歌单 ID 是歌单网页地址中 `id=` 后面的数字。播放器使用网易云官方外链，需要联网访问；歌单应设置为可公开访问。若嵌入播放器受浏览器或地区策略限制，可以使用播放器下方的“在网易云音乐打开”。

## 文件说明

- `index.html`：网页结构和各个弹窗。
- `styles.css`：全站视觉样式与响应式布局。
- `site-content.js`：欢迎页和个人资料默认内容。
- `site-posts.js`：学习笔记和杂谈文章。
- `script.js`：页面交互程序，日常更新内容通常不需要修改。

## 发布网站

这是纯静态网站，可以部署到 GitHub Pages、Cloudflare Pages、Netlify 等静态托管平台。发布前建议复制整个文件夹作为备份，并在电脑和手机尺寸下各检查一次。
